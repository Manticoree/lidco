## Быстрые ссылки

- **Режим CLI:** `lidco`
- **Режим сервера:** `lidco serve`
- **Порт по умолчанию:** `8321`
- **Конфигурация:** `~/.lidco/config.yaml` (глобальная) или `.lidco/config.yaml` (проектная)
